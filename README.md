# 豌豆AI工具

豌豆AI工具是一套 Windows 便携式 AI 创作工作台，包含节点画布、图片工作台、提示词工具和图片处理功能。

## 使用方法

1. 从 GitHub Releases 下载 `wandou-ai-tools-windows-x64.zip`。
2. 解压到任意可写文件夹。
3. 打开文件夹，双击带豌豆图标的 `豌豆AI工具.exe`。
4. 首次使用时在右上角设置 API 信息。

软件保留旧安装版的桌面窗口设计：顶部支持首页、多标签页、前进、后退、刷新、公告、版本号以及窗口按钮。它是普通文件夹便携版，不写安装信息，也不需要卸载。用户数据继续使用旧版的 `%APPDATA%\豌豆AI` 目录，覆盖升级不会清除工作流和设置。

## 自动更新

启动后程序会检查本仓库最新的稳定版 Release。发现新版时，软件顶部显示“发现新版本”：

1. 点击“有更新”查看更新说明。
2. 点击“立即更新”。
3. 程序直接下载便携 ZIP 并校验 SHA-256。
4. 校验成功后自动退出、覆盖程序文件并重新启动。

## 发布新版本

推送符合 `vX.Y.Z` 格式的标签后，GitHub Actions 会自动：

- 生成 Windows 便携 ZIP；
- 写入对应版本号；
- 生成 SHA-256 校验文件；
- 创建 GitHub Release 并上传下载文件。
