使用教程

豌豆AI 使用流程

按顺序完成 API 注册、密钥保存和图片生成。密钥只保存在本机浏览器里。

1

注册 API 平台账号

打开你的 API 平台网站，使用手机号、邮箱或微信方式注册并登录。进入控制台后，根据平台提示完成充值或开通模型权限。

2

复制 API 密钥

在 API 平台的“令牌管理 / API Key / 密钥管理”里创建密钥，复制以 sk- 开头的完整密钥。不要把密钥发给无关人员。

3

回到豌豆AI 填写密钥

点击首页右上角齿轮按钮，打开 API 设置，把复制的密钥粘贴进去并保存。保存后，绘图、关键词和文案工具会共用这份配置。

4

生成图片

回到首页，打开“工作台模式 AI 绘图”。输入提示词，也可以上传参考图；选择模型、尺寸、分辨率和数量后点击生成。
